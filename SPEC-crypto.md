# SPEC — `hashdigest` + `sealedbox`

Two small crypto leaf modules, thin over `std.crypto`. Two modules from one task. Wave P2 (AXP
extraction). Both `extract · any · util · reentrant`, deps **none (std only)**, portable.
**Seeds: `~/workspace/axp/axp-core/digest.zig` + `axp-core/task.zig` (`sha256FileHex`)** for
`hashdigest`; **`~/workspace/axp/axp-core/sealed.zig`** for `sealedbox` — same authors' Apache-2.0
code, relicensed MIT. Register `.{ .name = "hashdigest" }` and `.{ .name = "sealedbox" }`, no deps.

## `hashdigest`

Streaming SHA-256 helpers: `sha256Hex(bytes) [64]u8`, a streaming hasher (feed chunks → hex), and
**`sha256File`** that hashes a file in bounded chunks — importantly **works on size-0 virtual files**
(e.g. `/proc/*`) where `stat().size == 0` but the file has content (read until EOF, don't trust the
size). Content-address match helper (`matches(hex, bytes)`). Model after Go `crypto/sha256`
streaming. Thin over `std.crypto.hash.sha2.Sha256`.

- Tests: **known-answer vectors** (empty string, "abc", the standard SHA-256 test vectors) for
  one-shot AND streaming (chunked feed == one-shot); hex lowercase; `sha256File` on a temp file +
  on a size-0-reported source (read-to-EOF path); `matches` true/false.

## `sealedbox`

NaCl **`crypto_box_seal`** — anonymous-sender public-key encryption: encrypt to a recipient's
X25519 public key with **no sender key** (an ephemeral keypair per message; the recipient can't
identify the sender). `seal(recipient_pk, plaintext) → ciphertext` and `open(recipient_kp,
ciphertext) → plaintext`. The construction: ephemeral X25519 keypair; nonce = `blake2b(ephemeral_pk
|| recipient_pk)` (24 bytes); box = `crypto_box(plaintext, nonce, recipient_pk, ephemeral_sk)`;
output = `ephemeral_pk || box`. Thin over `std.crypto` (X25519 + XSalsa20-Poly1305 / the seed's
primitives — match the seed exactly). **Do NOT roll custom crypto** — compose std primitives as the
seed does. Model after libsodium sealed box / Go `nacl/box`.

- Tests: **seal→open round-trip** (random keypair, various plaintext sizes incl. empty); a **tampered
  ciphertext** (flip a byte in the box or the ephemeral pk) → `open` fails (auth); wrong recipient
  key → fails; the nonce-derivation is deterministic for a given (ephemeral_pk, recipient_pk).
  Include a **known-answer vector** if the seed has one, or one generated by libsodium (document the
  source); otherwise the round-trip + tamper tests are the gate. Assert the ciphertext length =
  `32 + plaintext_len + 16` (ephemeral pk + Poly1305 tag).
